# Problem 1: Print Numbers (5 points)

**Tier:** Trivial | **Est. time:** 1-2 minutes

## Specification
Print the integers from 1 to 1000, one number per line.

## Output
Save your output to: `problems/01-print-numbers/output.txt`

The file should contain exactly 1000 lines. The first line is `1`, the last line is `1000`. No leading zeros, no extra whitespace.
